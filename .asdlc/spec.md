# Overview

A URL shortener that lets people create short links through a web interface or API, with automatic cleanup of old links.

# Personas

- Link creator — submits long URLs and gets back short links to share.

# Features

- A person pastes a long URL and immediately gets a short link they can copy.
- Anyone who visits a short link gets redirected to the original long URL.
- The API accepts a long URL and returns a short link in JSON format.
- Short links that haven't been used in a while get automatically deleted.
- People can see basic stats about their short link, like how many times it's been clicked.
